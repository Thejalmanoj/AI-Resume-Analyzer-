# AI Resume Analyzer

AI-powered Resume Analyzer built using FastAPI, Streamlit, LangChain and FAISS.

## Features

- ATS Score Calculation
- Resume Parsing
- Skill Extraction
- Skill Gap Analysis
- Semantic Matching
- AI Feedback Generation

## Tech Stack

### Frontend

- Streamlit

### Backend

- FastAPI

### AI

- OpenAI
- LangChain
- FAISS
- Sentence Transformers

## Run Locally

### Backend

```bash
cd backend/app
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
streamlit run app.py
```
